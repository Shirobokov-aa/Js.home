alert - это инструкция

инструкции пишутся с новой строки и разделяются точкой с запятой ;

alert ('Пивет');
alert ('Мир');

в большенстве случаев точку с запятой ; можно не ставить, если есть переход на новую строку

"use strict" - строгий режим
когда он находится в начале скрипта, весь сцнарий работает в "современном режиме"


            ПЕРЕМЕННАЯ

Переменная задается инструкцией let

let massage;
message = 'hello';

alert(message);

так же можно написать сокращенно
let message = 'hello';

работать будет одинаково

переменная с числовм значение в ковычки  '' не заключается

let age;
age = 27;

alert(age);


Мы также можем объявить две переменные и скопировать данные из одной в другую.

let hello = 'Hello world!';

let message;

// копируем значение 'Hello world' из переменной hello в переменную message

message = hello;

// теперь две переменные содержат одинаковые данные
alert(hello); // Hello world!
alert(message); // Hello world!




чтобы обьявить константу (неизменяемую переменную) 
используют const вмето let

const myBirthday = '23.09.1995'

alert(myBirthday);


ИТОГ ПЕРЕМЕННОЙ

let - современный способ обьявления ПЕРЕМЕННОЙ

var - устаревший способ

const - похожее на let, но значение переменной не может изменяться
